
# QSupportViewer

QSupportViewer(仮称)は、音楽、画像、動画の管理と再生を一つのアプリケーションに統合した、多機能メディアビューアです。 

## ダウンロード
**現在準備中です。**

## インストール・実行方法
1. 配布されている zip ファイルを解凍してください。
2. 解凍したフォルダ内の `QSupportViewer.exe` をダブルクリックして起動します。

※ フォルダ内の `.dll` ファイルなどは削除したり移動したりしないでください。起動できなくなります。

## 動作環境
* OS: Windows 10 / 11 (64bit)
